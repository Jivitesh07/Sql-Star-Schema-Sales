# Sql-Star-Schema-Sales

## Project Overview
This project demonstrates **SQL data modeling** by building a **Star Schema** for a retail sales dataset.  
The goal is to transform raw transactional data into a structured **data warehouse** optimized for analytics and reporting.

## Tools & Technologies
- MySQL / PostgreSQL  
- SQL  
- CSV  
- draw.io / dbdiagram.io (for schema design)

## Schema Design
- **Fact Table**: `fact_sales`
- **Dimension Tables**:
  - `dim_customer`
  - `dim_product`
  - `dim_region`
  - `dim_date`

## Key Analytics
- Region-wise total sales  
- Monthly sales trends  
- Top customers by revenue  
- Data validation using record counts

## Key Learning Outcomes
- Star schema data modeling  
- Fact & dimension table design  
- SQL joins and indexing  
- Analytical query optimization
